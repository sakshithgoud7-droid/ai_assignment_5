# AI Assignment 5

## Overview

This assignment demonstrates the implementation of core Artificial Intelligence concepts using Python. It includes game search algorithms, a travel recommendation system, a knowledge graph, and a Bayesian network.

## Components

### 1. Tic-Tac-Toe AI

Implementation of:

* Minimax Algorithm
* Alpha-Beta Pruning
* Heuristic Search
* Monte Carlo Tree Search (MCTS)

**Features**

* Intelligent move selection
* Game state evaluation
* Search optimization

### 2. Travel Planner

A simple recommendation system that suggests destinations based on:

* Budget
* Travel interests

**Destinations Included**

* Goa
* Manali
* Jaipur

### 3. Knowledge Graph

Represents relationships between AI-related concepts using graphs.

**Concepts**

* Python
* Artificial Intelligence
* Machine Learning
* Deep Learning
* Data Science

### 4. Bayesian Network

A probabilistic model for traffic prediction using:

* Rain
* Accident
* Traffic

## Requirements

* Python 3.10+
* networkx
* matplotlib
* pgmpy

Install dependencies:

```bash
pip install networkx matplotlib pgmpy
```

## Run

```bash
python tic_tac_toe.py
python travel_planner.py
python knowledge_graph.py
python bayesian_network.py
```

## Output

* AI-generated game decisions
* Travel recommendations
* Knowledge graph visualization
* Traffic probability predictions

## Conclusion

This assignment provides practical experience with AI techniques including game search, recommendation systems, knowledge representation, and probabilistic reasoning.
