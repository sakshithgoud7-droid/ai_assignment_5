# 🤖 AI Assignment 5

This repository contains the implementation of various Artificial Intelligence concepts using Python. The assignment covers game search algorithms, recommendation systems, knowledge representation, and probabilistic reasoning.

---

## 📌 Contents

### 🎮 1. Tic-Tac-Toe AI
Implementation of multiple AI search techniques for playing Tic-Tac-Toe.

Algorithms Used:
- Minimax
- Alpha-Beta Pruning
- Heuristic Search
- Monte Carlo Tree Search (MCTS)

Features:
- Intelligent move selection
- Board state evaluation
- Search tree optimization
- Simulation-based decision making

---

### ✈️ 2. Travel Planner
A simple AI-based travel recommendation system.

Features:
- Budget-based destination selection
- Interest matching
- Food recommendations
- Suggested travel duration

Destinations:
- Goa
- Manali
- Jaipur

---

### 🕸️ 3. Knowledge Graph
A basic knowledge graph demonstrating relationships between AI concepts.

Concepts Included:
- Python
- Artificial Intelligence
- Machine Learning

- Data Science

Libraries:
- NetworkX

### 📊 4. Bayesian Network
A probabilistic model for traffic prediction.

Variables:
